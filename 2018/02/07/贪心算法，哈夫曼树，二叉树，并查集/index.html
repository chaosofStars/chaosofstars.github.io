优先队列（贪心算法）

概念

PriorityQueue（优先队列）在Java 中底层实现用的是小根堆，所以在具体问题上需要自己构建比较器来实现不同的优先队列。

题目1:

随时找到数据流的中位数
【题目】
有一个源源不断地吐出整数的数据流，假设你有足够的空间来
保存吐出的数。请设计一个名叫MedianHolder的结构，
MedianHolder可以随时取得之前吐出所有数的中位数。
【要求】
1．如果MedianHolder已经保存了吐出的N个数，那么任意时刻
将一个新数加入到MedianHolder的过程，其时间复杂度是
O(logN)。
2．取得已经吐出的N个数整体的中位数的过程，时间复杂度为
O(1)。

思路：

构建一个大根堆和一个小根堆，维持平衡，根据Size大小，弹出大根堆或小根堆堆顶。

题目4：

折纸问题
【题目】
请把一段纸条竖着放在桌子上，然后从纸条的下边向上方对折1次，压出折痕后展开。此时
折痕是凹下去的，即折痕突起的方向指向纸条的背面。如果从纸条的下边向上方连续对折2
次，压出折痕后展开，此时有三条折痕，从上到下依次是下折痕、下折痕和上折痕。给定一
个输入参数N，代表纸条都从下边向上方连续对折N次，请从上到下打印所有折痕的方向。

down
N=2时，打印：
down
down
up

实质是一个n层的二叉树，父节点和左子树向下，右子树向上。

题目5：

二叉树先序、中序，后序遍历的非递归实现

思路：

  1. 非递归的前序遍历和中序遍历都靠压栈。区别是，一个为先打印父节点，后压栈，另一个是遍历左子树，打印，返回，打印右子树。
  2. 非递归后序遍历有两种思路：一种是通过两个栈，一个收集节点，然后倒出来，交给另一个栈，最后打印。另一种是遍历左子树，右子树，记录遍历过的信息，最后返回。

题目6：

在二叉树中找到一个节点的后继节点
【题目】
现在有一种新的二叉树节点类型如下：
public class Node {
public int value;
public Node left;
public Node right;
public Node parent;
public Node(int data) {
this.value = data;
}
}

该结构比普通二叉树节点结构多了一个指向父节点的parent指针。假设有一
棵Node类型的节点组成的二叉树，树中每个节点的parent指针都正确地指向
自己的父节点，头节点的parent指向null。只给一个在二叉树中的某个节点
node，请实现返回node的后继节点的函数。在二叉树的中序遍历的序列中，
node的下一个节点叫作node的后继节点。

思路：

分几种情况：

  1. 当前点有右子树时，后继结点为当前节点的右子树的最左端。
  2. 当前点没有右子树时，后继结点为第一个为左子树的父节点，如果没有则为null。

并查集：

概念：

并查集有几个集合组成，每个集合由若干个节点，满足以下几个条件：

  1. 并查集可以快速合并两个节点所在的集合。
  2. 并查集可以快速查询某个节点是否属于某一集合。
  3. 当数据量很大时，并查集时间复杂度可以降到O（1）。

另外一点比较有趣的是，并查集这个算法史上可以名列前五的算法，发现的很早，但是证明它的时间复杂度证了25年。
